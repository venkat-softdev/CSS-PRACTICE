CSS TEXT

Introduction:

  • css text properties are used to controls the appreance ,layout ,and spacing of textual content on a web page.
  • they provide a wide range of options to style text acoording to design

    • KEY CSS TEXT PROPERTIES :

  • color: sets the color of the text.
  • text-allign: controls the horizontal alignment of text (eg: left,right,center).
  • text-decoration: Applies decoration to text such as underline ,overline , line-through.
  • text-transform: controls text capitalization ,like uppercase,lowercase,capitalize.
  • line-height: sets the amount of space between lines of text.
  • letter-spacing: controls the spacing between charcters
  • word-spacing: controls the spacing between words
  • font-family: defines the typeface to be used.
  • font-size: controls the sets of the text
  • font-weight: sets the thickness of characterts (eg: normal,bold)
  • font-style: Applies styles like italic or oblique.
  • font: A shorthand property to set multiple font properties at once .

    • 1. Text color,size ,and font family:

  • use the color property to set color of the text.
  • use the font-size property to set the size of the text.
  • use the font-family property to set the font family of the text.
