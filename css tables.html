CSS TABLES

introduction

  • css tables are used to style and format html tables, enhancing their appearance and functionally.
  • they are provide control over table layout ,spacing ,border, colors and responsives.

    • key css table properties

    • border: define the style ,width, and color of table border.
    • padding: controls the space between cell content and its border.
    • text allign: sets the horizontal alignment of text within cells.
    • vertica-allign: controls the vertical alignment of content within cells.
    • width/height: sets the dimensions of the table or indivial cells.
    • border-collapse: determines whether table border are collapsed into a single border or separated.
    • background-color: applies background color to the table or specific cells.
