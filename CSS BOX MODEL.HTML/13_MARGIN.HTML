CSS MARGIN

introduction:

  • css margin are used to create space elements ,outside of any defined borders.
  • they play a vital role in controliing the layout and spacing between elements on a web page.

    • Key css margin properties:

    • margin-top: sets the top margin of an element.
    • margin-right: sets the right margin of an elements .
    • margin-bottom: sets the bottom margin of an elements.
    • margin-left: sets the left margin of an elements.
    • margin: A shorthand to set all four margins at once

    2. Applying margins

    the margin property canb be applied in several ways

    • margin:value : Applies the same margin to all sides of the element.
    • margin: top right bottom left: Applies different margins to ecah side of the element in clockwiseorder.
    • margin-top: value: Applies a specific margin to the top of the element.
    • margin-bottom:value: Applies a specific margin to the bottom of the element.
    • margin-left:value: applies a specific margin to the left of the element.
    • margin-right:value; Applies a specific margin to the right of the element.
    • margin: vertcical horizontal: Applies diiferent to /bottom and left/right margins.
    • margin:auto; centers the element horizontally by setting left and right margin to auto.
