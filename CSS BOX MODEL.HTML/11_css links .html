CSS LINKS

Key properties for links

  • color: sets the color of the link text.
  • text-decoration: controls the decoration (eg: underline) of the link.
  • font-weight: sets the thickness of the link text.
  • background-color: Applies a background color to the link.
  • :link:target unvisited links.
  • :visited: targets visited links.
  • :hover: targets links when the mouse pointer is over them.
  • :active: targets active links ,such as during a click.
