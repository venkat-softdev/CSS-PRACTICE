CSS LISTS

Introduction:

  • css lists are used to style ordered (ol),unordered(ul),and description(dt) in html.
  • they allow for customization of list markers , layout, and appreance.

    • key css list properties:

  • list-style-type: specifies the type of list marker (eg: disc,circle,decimal,etc...)
  • list-style-position: detrmines the position of the liat marker (inside or outside the content flow).
  • list-atyle-image: sets an image as the list marker.
  • list-style: a shorthand properyty to set type ,position,and image all at once.
