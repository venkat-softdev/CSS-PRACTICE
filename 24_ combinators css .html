Types of css combinators:

  • Descendent combinators (space): Targets all Descendent of a specified element.
  • Child Combinator (>): Targets all immediate children of a specified element.
  • Adjacent sibling combinator(+): Targets the immediate next sibling of a specified elements.
  • General sibling combinator(-): targets all siblings that share the same parent as the specified elements.
