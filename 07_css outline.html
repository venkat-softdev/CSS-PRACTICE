CSS OUTLINE

Introduction

  • css outlines are lines drawn around elements ,outside the border edge
  • unlike borders ,outlines do not take up space and are often used for accessbility.
  • outline-style: defines the style of the outline ,such as 'solid','dashed','dotted',etc.
  • outline-width: sets the thickness of the outline.
  • outline-color: specifies the color of the ouline
  • outline-offset: controls the distance between the outline and the border edge.
  • outline: A shorthand property to set the style ,width,and color all at once.
