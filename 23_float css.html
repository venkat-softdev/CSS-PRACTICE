Key css float properties

  • float:left; float the elements to the left side of its container.
  • float:right; float the elements to the right side of its container.
  • float:none; resets the float property mainatining the normal document flow.
  • clear: controls how elements following a floacted elements behave (eg; claer:both; prevents elements from flaoting on both sides.)
