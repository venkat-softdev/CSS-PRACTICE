CSS Z-Index

Introduction:

  • the css Z-index property controls the vertical shocking order of elemnts that overlap.
  • it's used in conjuction with the position (absolute,relative fixed or sticky ) to determine which eleements appear in front or behind others.

    • How its Z-index works

    • elements with a higher z-index value are renbderd in front of those with a lower value .
    • if z-index is not specified elements follow the natural stacking order based on the z-index in the html eleemnts.
    • it only affects elements with a position value other than static.
