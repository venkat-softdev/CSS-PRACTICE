CSS POSITION

Introduction:

  • css position properties are used to control the positioning of elements with the layout of web page.
  • they provide flexibility in aligning elemnts relative to their parent container or the view port .

    • key css position properties:

    • static: the default value,where eleemnts are positioned according to the normal flow of the document.
    • relative: positions the eleemnts relative to its normal position ,without affecting other elements.
    • absolute: positions the elements ralative to the nearest positioned ancestor (other than static).
    • fixed: positions the elements relative to the view port ,so its stays in the same place even if the page is scrolled.
    • sticky : behaves like a combination of realtive and fixed ,sticking to the view port within a certain area.
