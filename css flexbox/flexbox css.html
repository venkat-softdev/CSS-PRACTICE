Css Flexbox

introduction:

  • css flexbox or the flexible box layout ,is a layout model that allows easy design of complex layouts with alignment capabilities.
  • it provides an efficient way to distribute space and align items within complex layouts and when the size of items in unknown or dynamic.

Key concepts:

  • flex container: the parent elements that holds flex items
  • flex items: the child elements within a flex container.
  • main axis: the primary axis along which flex items are laid out.
  • cross axis: the axis perpendicular to the main axis.

    • flexbox properties:

  • 1.display:defines a flex container(eg;display:flex)
  • 2. flex-direction: defines the direction of the main axis.
  • 3.justify-content: aligns items along the main axis.horizontal alignment in x-axis.
  • 4.align-items: align items along the cross axis. vertical alignment in y-axis.
  • 5.align-self: property is used to control the alignment of individual flex items within a flex container.
  • 6.flex grow,flex-shrink,flex-basis:control how flex items grow and shrink within the container.
