syntax and usage of opacity

  • the opacity property is ued to define the transparency level of element.
  • it allows designers to craete overlays ,fade effects, and other visual effects.
  • the opacity property takes values fom 0to1,where 0 is fully transparent and 1 is full opaque.
  • example: opacity:0.5; will render the elements with 50% transparency.

1.changing elements opacity:

  • use the opacity property to adjust the transparent of an elements.
  • the values ranges from 0 (fully transparent) to 1 (fully opaque)

    • 2.Applying opacity on text

    • opacity can also be applied to text inside an elements.
    • the opacity value of the text affects the text and any child elements.
