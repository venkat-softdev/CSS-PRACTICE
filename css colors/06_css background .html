CSS BACKGROUND

key background properties:

  • background-color: sets the background color of an element.
  • background-image: specifies one or more background imaGES For an element.
  • backgrpund-size: defines the size of the background images.
  • background-position: specifies the position of the background iamges .
  • background-repeat: sets if/how background images will be repeated.
  • background-attachment: determine whether the background images scrolls.

    4. Background Size:

    • the background-size property specifies the size of the background images.
      • auto: default the background image relations its original size.
      • cover: scales the background images to cover the entiore container,possibly cropping it.
      • contain: sacles the background images to be fully visible futting inside the conatiner (may leave empty space)
      • length or percentage: set an explict width and /or height (eg: 100px,80px,or 50)
      • multiple,comma-separates: you can set different sizes for multiple background images.
