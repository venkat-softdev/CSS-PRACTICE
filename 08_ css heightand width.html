CSS Height and Width

Introduction

  • the css properties 'height' and 'width' are used to control the dimensions of html elements.
  • they are essential for defining the layout and appreance of elements on a web.

    • Key properties :

  • height: sets the height of an elements can be defined using pixels,percentage ,em units etc.
  • width; srts the width of an elements .can be defined using similar units as height.
  • min-height/max-height: define the minimum and maximum height for an elements.
  • min-width/max-width: define the minimum and maximum width for an element.
