Key css math functions:

  • calc(): calculates a value based on a mathematical expression,eg:calc(100%-50px)
  • min(): returns the smallest value from set of values eg, min(100px,50%)
  • max(): returns the largest values from a set of values max(2rem,50%)
  • clamp():clamps a values between a minimum and maximum value (clamp 20px /50%,200px)
